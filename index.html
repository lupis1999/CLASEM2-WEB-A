Selectores basicos

Los selectores básicos en CSS permiten seleccionar elementos HTML para aplicarles estilos.

  • Selector de tipo:(etiqueta)selecciona elementos por su nombre de etiqueta. Ejemplo: ppara seleccionar todos los párrafos.
  • Selector de clase:selecciona elementos por su atributo de clase. Ejemplo: .mi-clasepara selecionar todos los elementos con la clase ¨mi-clase¨.
  • Selector de ID: selecciona un elemento único por su atributo de ID. Ejemplo: #mi-idpara seleccionar todos los elementos con el ID "mi-id".

Selector universal y combinaciones

El selector universal (*) selecciona todos los elementos en la página.Ejemplos Incluyen: * { margin: 0; padding: 0; }

Los selectores combinados permiten seleccionar elementos basados en su relación con otros elementos. Ejemplo: div p { color: red; } (selecciona todos los párrafos dentro de un div)

  • selector universal:selecciona todos los elementos por ejemplo,*.
  • selectores de hijos:selecciona elementos hijos directos de otro elemento. Ejemplo: div > p.
  • selectores de descendientes:selecciona elementos descendientes de otro elemento. Ejemplo: div p.

Display y tipos de cajas

El atributo display define cómo se comporta un elemento en el layout de la página.

  • block: elementos que ocupan todo el ancho disponible.
  • inline: elementos que se colocan uno al lado del otro.
  • inline-block: elementos que se comportan como inline pero pueden tener dimensiones.

Pseudo-clases

Las pseudo-clases son selectores que permiten aplicar estilos a elementos en ciertos estados.

  • :hover: aplica estilos cuando el usuario pasa el cursor sobre un elemento.
  • :focus: aplica estilos cuando un elemento tiene el foco (por ejemplo, un campo de entrada).
  • :active: aplica estilos cuando un elemento está activo (por ejemplo, al hacer clic en un botón).
