Selectores basicos

Los selectores básicos en CSS permiten seleccionar elementos HTML para aplicarles estilos.

  • Selector de tipo:(etiqueta)selecciona elementos por su nombre de etiqueta. Ejemplo: p { color: blue; }
  • Selector de clase: selecciona elementos por su atributo de clase. Ejemplo: .mi-clase { font-size: 18px; }
  • Selector de ID: selecciona un elemento único por su atributo de ID. Ejemplo: #mi-id { background-color: yellow; }

Selector universal y combinaciones

El selector universal (*) selecciona todos los elementos en la página. Ejemplo: * { margin: 0; padding: 0; }

Los selectores combinados permiten seleccionar elementos basados en su relación con otros elementos. Ejemplo: div p { color: red; } (selecciona todos los párrafos dentro de un div)
